Detalhes Técnicos

O Problema e a Solução

Em situações onde a internet é bloqueada, precisamos de formas alternativas de comunicação. O Protocolo NOISE utiliza a esteganografia (arte de esconder mensagens) em ficheiros de áudio.

Escolhemos o Heavy Metal e Punk porque a distorção natural das guitarras cria "ruído" suficiente para escondermos dados encriptados sem que o ouvido humano perceba a diferença.

Metodologia

  • Linguagem: C (para performance máxima).
  • Criptografia: AES-256 antes da injeção no áudio.
  • Frequência: Dados inseridos acima de 18kHz.
  • Plataforma: Desenvolvido e testado em Arch Linux.

O algoritmo analisa o ficheiro .WAV e procura picos de volume para inserir os bits da mensagem secreta.

Excerto do Código (C)

Exemplo simplificado da função de injeção:

void esconder_dados(char *audio, char *msg) {
    int i;
    for(i = 0; i < tamanho_buffer; i++) {
        // Se detetar distorção alta
        if(audio[i] > LIMIT_DISTORTION) {
            inject_bit(audio[i], msg);
        }
    }
    printf("Dados injetados com sucesso.\n");
}
